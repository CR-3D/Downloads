# CR-3D Downloads

Willkommen im **CR-3D Downloads** GitHub-Repository.  
Hier findest du alle wichtigen **Downloads** rund um CR-3D – übersichtlich nach Kategorien sortiert: **Filamente**, **technische Datenblätter**, **Sicherheitsdatenblätter (SDS/MSDS)**, Produktinfos und weitere Dokumente.

## Inhalte

- **Filamente**
  - Produktübersichten
  - Druckempfehlungen / Parameter (wenn verfügbar)
  - Farb- und Materialinformationen

- **Technische Datenblätter (TDS)**
  - Materialkennwerte
  - Verarbeitungshinweise
  - Anwendungsbereiche

- **Sicherheitsdatenblätter (SDS / MSDS)**
  - Gefahrenhinweise
  - Lagerung & Handhabung
  - Entsorgung / Transport

- **Weitere Downloads**
  - Produktbroschüren / Flyer
  - Zertifikate (falls vorhanden)
  - Anleitungen / Guides

## Hinweise

- Die Dokumente werden regelmäßig aktualisiert.  
- Maßgeblich ist immer die **neueste Version** im jeweiligen Ordner.
- Sicherheitsdatenblätter sind – je nach Hersteller – in mehreren Sprachen verfügbar.

## Kontakt / Support

Wenn dir ein Dokument fehlt oder du eine aktualisierte Version brauchst, melde dich bitte bei **CR-3D** (Support/Vertrieb) oder erstelle ein **Issue** in diesem Repository mit:
- Material / Produkt
- gewünschtem Dokumenttyp (TDB/SDB/etc.)
- optional: Artikelnummer oder Link zum Produkt
